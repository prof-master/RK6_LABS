В директории предоставлены нсеколько вариантов выполненимя поставленной задачи.
Файл с название lower_сase воспринимает шестнадцатиричные выражения с маленьких букв.
Файл с название upper_сase воспринимает шестнадцатиричные выражения с больших букв.
Файл с название any_сase воспринимает шестнадцатиричные выражения как с больших букв, так и с маленьких.
